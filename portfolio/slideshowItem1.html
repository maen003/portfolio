The purpose of this application is to allow students to create study groups in their area and also allow other students to join existing study groups.

The visible elements of the front end were built in React and Redux, a JavaScript framework that allows elements on the page to render without refreshing

This application communicated with a server built using Node.js and Express.js and communicated with the Facebook API in order to authenticate users

Users and events were stored in a MySQL database which communicated with the Node.js server
