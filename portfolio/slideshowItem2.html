Culinary Generator

Slideshow

The purpose of this application was to build an app that would suggest a restaurant based on the most popular food genres on Yelp.

This Yelp generator was built using core JavaScript as well as jQuery, a JavaScript framework to communicate with Yelp's API and Google Maps' API

AJAX calls were used to make requests to Yelp's and Google's respective APIs in order to display data based on the searched results
